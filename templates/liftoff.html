Description

A track showing gene models as from liftoff.

Methods

Display Conventions and Configuration

Liftoff was run with the following command using the comprehensive GenCode Release 35 (GRCh38.p13) gff. liftoff -dir {output.temp} -sc 0.85 -copies -p {threads} -g {input.gff} -o {output.gff} -u {output.unmapped} {input.t} {input.r} Custom scripts were used to make a bed file from the resulting gff.
