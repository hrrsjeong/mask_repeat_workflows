Description

A track showing ancestral segmental duplications as generated by DupMasker on the assembly: {SM}.
The fourth column is a name of the inferred ancestral copy of the SD and the 9th column shows a color that reflects the first column.

Methods

DupMasker was run with default parameters on a RepeatMasked genome assembly. The resulting output was then converted into a browser friendly format (bed9).

Display Conventions and Configuration

Standard colored bed9 file with RGB values in the 9th column.
